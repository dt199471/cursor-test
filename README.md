# 可愛いオセロゲーム 🎮

子供でも楽しく簡単に遊べるような「可愛さ」を取り入れたレイアウトのオセロゲームです。

## プロジェクト概要

このプロジェクトは、Codex、Claude、Geminiの3つのAIモデルでそれぞれ異なるアプローチで実装されたオセロゲームの比較プロジェクトです。

## ブランチ構成

- `codex`: Codexで実装されたバージョン
- `claude`: Claudeで実装されたバージョン
- `gemini`: Geminiで実装されたバージョン

## 要件

- 子供でも楽しく遊べる
- 可愛いデザイン
- 簡単な操作
- 直感的なUI

## 開発環境

各ブランチで異なる実装アプローチを試しています。
